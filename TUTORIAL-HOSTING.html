Tutorial Hosting Game "Dunia Baca" di GitHub Pages

Panduan Lengkap + Gambar

📋 Ringkasan 4 Langkah

  1. Buat repository & upload file — 2 menit
  2. Aktifkan & jalankan GitHub Actions — 1 menit
  3. Setting GitHub Pages ke branch gh-pages — 30 detik
  4. Buka game! 🎉
⚠️ Kenapa tadi layar putih?
GitHub Pages menampilkan file langsung dari branch main. Tapi file di main itu source code (TypeScript/React) yang tidak bisa dibaca browser. Harus di-build dulu menjadi HTML+JS biasa. Itulah tugas GitHub Actions — dia build otomatis dan taruh hasilnya di branch gh-pages.

Checklist Sebelum Buka Game

Pastikan semua ini sudah benar

  • Folder .github/workflows/ ada di repository
  • File deploy-pages.yml ada di dalam folder tersebut
  • Tab Actions: workflow sudah berjalan dan ✅ hijau
  • Branch gh-pages sudah muncul di daftar branch
  • Settings → Pages → Branch = gh-pages (BUKAN main)
  • Folder node_modules TIDAK di-upload
